# Database-Student-Management-System

The **Student Management System** is a role-based web application built with ASP.NET MVC. It enables administrative users to efficiently manage student data, ensuring secure access, organized records, and easy reporting.

## Features

- 🛡️ Secure admin login with role-based authorization
- 📋 Add, edit, delete, and view detailed student profiles
- 📷 Upload and display student profile photos
- 🔍 Search students by first or last name
- 📊 Generate and download student reports (HTML and CSV)
- ✅ Clean and responsive user interface

## Technologies Used

- ASP.NET MVC 5
- Entity Framework (Code First)
- SQL Server
- HTML5, CSS3
- C# (Server-side)

## How to Run the Project

1. Clone the repository to your local machine.
2. Open the solution in Visual Studio.
3. Set up your database connection string in `Web.config`.
4. Run the application using IIS Express or your preferred server.
5. Log in as an admin to access full functionality.

## Report Access

- View HTML Report:  
  `http://localhost:[port]/Admin/StudentReport`

- Download CSV Report:  
  `http://localhost:[port]/Admin/GenerateStudentReport`

## Admin Functionalities

- Full student record management
- Photo upload and update
- Export reports in CSV format
- Profile-based viewing and search
